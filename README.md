# NUXT 3 Sample App fro Frentend Course

---
A simople TODO app implemnting the following texchnologies:
- NUXT 3
- Typescript
- Pinia
- MongoDB and Mongosh

## Setup

To install the dependencies:

```bash
pnpm install
```

## Development Server

Start the development server on http://localhost:3001

```bash
pnpm run dev
```
